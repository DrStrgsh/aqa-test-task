# BG-003 500 error is displayed instead of 404 for non-existing Project pages

## Description:
- **500** error is displayed instead of **404** for non-existing **Project** pages

## Severity:
- **Minor**

## Environments:
- Any

## Preconditions:
- User is logged in
- User is located on projects page (https://staging-stack.alynea.io/projects)

Steps:
1. Add non-existing **Project Id** or any text to the end of URL (ex: '/sometext')
2. Go to the page

## Expected result:
- **404** Error page is displayed

## Actual result:
- **500** Error page is displayed

- [bg-003-image](https://prnt.sc/MAy_twB91RG8)
- DevTools logs: GET https://staging-stack.alynea.io/projects/sometext **500** (Internal Server Error)
