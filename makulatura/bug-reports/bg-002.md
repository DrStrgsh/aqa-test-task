# BG-002 User can create project with name longer than 300 characters

## Description:
- **ProjectName** input field does not have any length validation

## Severity:
- **Trivial**

## Environments:
- Any

## Preconditions:
- User is logged in
- User is located on **Create Project** page (https://staging-stack.alynea.io/projects/new)

Steps:
1. Fill all fields except **'Name'** with valid data
2. Fill the **'Name'** field with more than 300 characters
3. Click the **'Create Project'** button

## Expected result:
- Validation error for the **Name** field is displayed

## Actual result:
- No validation error is displayed and project succesfully created
