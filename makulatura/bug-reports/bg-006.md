# BG-006 'City not found' Flash message appears when wrong URL is entered

## Description:
- **'City not found'** Flash message appears when wrong URL is entered

## Severity:
- **Trivial**

## Environments:
- Any

## Preconditions:
- User is logged in

Steps:
 1. Add any text to the end of URL (ex: '/sometext')
 2. Go to the page

## Expected result:
- **404** Error page is displayed

## Actual result:
- Flash message with text **'City not found'**
