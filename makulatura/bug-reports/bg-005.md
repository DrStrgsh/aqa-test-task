# BG-005 Mobile: 'Create Project' and 'Cancel' buttons are overlapped with the footer

## Description:
- **'Create Project'** and **'Cancel'** buttons are overlapped with the footer

## Severity:
- **Minor**

## Environments:
- Mobile (Pixel 5 & iPhone 12)

## Preconditions:
- User is logged in
- User is located on Create Project page (https://staging-stack.alynea.io/projects/new)

Steps:
1. Scroll down at the end of the page

## Expected result:
- **'Create Project'** and **'Cancel'** buttons are fully visible

## Actual result:
- **'Create Project'** and **'Cancel'** buttons are overlapped with the footer
