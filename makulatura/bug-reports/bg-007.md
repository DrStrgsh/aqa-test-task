# BG-007 The **Address** field automatically selects the first address from the list, even when it is incorrect.

## Description:
- The **Address** field automatically selects the first address from the list, even when it is incorrect. This causes the **Map** to behave incorrectly. The **Map Marker** is placed in the wrong location, the **Map** starts freezing, and the **Address** field temporarily stops suggesting new location options if the user starts typing again.

## Severity:
- **Minor**

## Environments:
- Any

## Preconditions:
- User is logged in
- User is located on Create Project page (https://staging-stack.alynea.io/projects/new)

Steps:
 1. Start typing a **non-existent city** or **any random sequence of letters** in the **Address** field (ex: `sometext`).
 2. Clear **Address** field.
 3. Repeat ***Step 1***

## Expected result:
- Nothing was selected. The **Map Marker** did not appear. The **Map** does not change its zoom level.


## Actual result:
- The **Address** field automatically selects the first address from the list, The **Map Marker** is placed on the first city that matches the initial letters. The **Map** changes its zoom level and focuses on the **Map marker**.
